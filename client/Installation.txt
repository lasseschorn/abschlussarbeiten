erstmalige Installation: npm install -g @angular/cli (am besten im client-Ordner)

in client: npm install (erstellt node_modules)
in client: ng build (erstellt dist-Ordner)
in server: Inhalt aus dem dist-Ordner(im client) in den server/assets Ordner kopieren
in XAMPP: Module Apache und MySQL starten
in server: npm start
in client: npm start
